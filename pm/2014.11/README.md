>版本控制是对软件开发进程中文件或目录的发展过程提供有效的追踪手段，保证在需要时可回到旧的版本，避免文件的丢失、修改的丢失和相互覆盖，版本控制在空间上可以保证完成集中统一管理，解决一致性和冗余问题。具体本案例来说可以实现如下功能
